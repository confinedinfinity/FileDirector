# FileDirector

**大量の動画・画像ファイルを簡単に整理・管理するツール**

---

## 📌 こんな悩み、ありませんか？

- 📂 **動画ファイルがフォルダにバラバラ** → 探すのに毎回時間がかかる
- 🔍 **「あの面白い動画、どこだっけ？」** → ファイル名を思い出せない
- ⭐ **お気に入り動画を整理したい** → タグや評価で分類したい
- 🎬 **中身を見ないと分からない** → サムネイルで一覧表示したい
- 📥 **ダウンロードした動画が散乱** → 自動で分類してほしい
- 🔄 **同じファイルが複数ある** → 重複を自動検出したい

**DirectorizerQtは、これらの問題をすべて解決します。**

---

## ✨ DirectorizerQtでできること

### 1. 📁 **動画・画像を一箇所で管理**
フォルダに散らばったファイルをデータベースに登録。
サムネイル付き一覧表示で、探したいファイルが一目で見つかります。

### 2. 🏷️ **タグ・評価で整理**
- お気に入り動画に「★★★★★」評価
- 「面白い」「勉強用」「旅行」などカスタムタグで分類
- タグや評価で瞬時に絞り込み検索

### 3. 🎬 **ブラウザで動画再生**
わざわざ別のプレイヤーを開かなくても、ブラウザでそのまま再生。
キーボードショートカットで快適操作（Space:再生/停止、←→:シーク、Ctrl+T:タグ編集）。

### 4. 📥 **YouTube・SNSから自動ダウンロード**
- YouTube動画を高画質でダウンロード
- Twitter・Instagram・Pixivの画像・動画を一括取得
- ダウンロード完了したら自動でデータベースに追加

### 5. 🔍 **重複ファイル・移動したファイルを自動検出**
- 同じファイルが複数ある場合を検出
- フォルダを移動したファイルを自動追跡
- ファイル名が変わっても内容で検索

### 6. 🔄 **動画変換・音声抽出**
- 動画形式を簡単に変換（MP4→AVI、圧縮など）
- 動画から音声だけ抽出
- 複数ファイルを一括処理

## 🚀 5分で始める

### **ステップ1: ダウンロード**
ZIPファイルを解凍して、`QtGuiApplicationTrial.exe` をダブルクリック。

### **ステップ2: データベース作成**
1. 初回起動時にウェルカム画面が表示されます
2. 「新規データベース作成」を選択
3. 名前を入力（例: 「動画コレクション」）
4. 保存場所を選択

### **ステップ3: ファイルを登録**
1. 「File」メニュー → 「Add Directory」
2. 動画・画像が入っているフォルダを選択
3. 自動でサムネイル生成 → 一覧表示

### **ステップ4: タグ付け・整理**
1. ファイルを右クリック → 「Edit Tags」
2. タグを追加（例: 「お気に入り」「旅行」）
3. 評価を設定（★1～5）

**これで完了！** あとは検索・フィルターで快適にファイルを探せます。

→ 詳しくは [5分クイックスタート](QUICKSTART.md) をご覧ください

- 重要度で評価・検索
- Web UIでスマホ・タブレットから視聴

## 🔧 動作環境

- **Windows 10 / 11** (64-bit)
- **メモリ**: 4GB以上推奨
- **ディスク容量**: 100MB以上

**外部ツール（オプション）**:
- **FFmpeg** - 動画変換・サムネイル生成に必要
- **yt-dlp** - YouTube動画ダウンロードに必要
- **Gallery-DL** - Twitter・Instagram等ダウンロードに必要

→ 外部ツールの設定方法は [ユーザーガイド](USER_GUIDE.md#外部ツール設定) をご覧ください

## 📖 用語説明

- **データベース（DB）**: ファイル情報を保存する場所。複数作成できます（例: 「動画用」「画像用」）
- **タグ**: ファイルに付ける目印（例: 「お気に入り」「仕事用」「旅行」）
- **サムネイル**: 動画・画像の縮小プレビュー画像
- **File Finder**: ファイルを検索・重複検出する機能
