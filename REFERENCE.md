# リファレンス

ショートカット、サポートファイル形式、動作環境の一覧です。

---

## キーボードショートカット

ショートカットは**メディアコントロールパネル**（プレイヤー下部）か**動画ウィンドウ**にフォーカスがある状態で動作します。
`Ctrl+E`（タグ入力欄へ）/ `Ctrl+M`（プレイヤーへ）でフォーカスを切り替えられます。

### 再生制御（動画ウィンドウにフォーカス）

| キー | 機能 |
|------|------|
| **Space** | 再生 / 一時停止 |
| **←** | 前にシーク（数秒戻る） |
| **→** | 後にシーク（数秒進む） |
| **Escape** | 静音・フォーカス解除 |
| **0** | 先頭へジャンプ |
| **1**〜**9** | 再生位置を10%〜90%にジャンプ |

### ナビゲーション・チャプター（メディアコントロールパネルにフォーカス）

| キー | 機能 |
|------|------|
| **Ctrl+N** | 次のファイルへ |
| **Ctrl+P** | 前のファイルへ |
| **Ctrl+F** | 1項目進む |
| **Ctrl+B** | 1項目戻る |
| **Ctrl+→** / **Shift+→** | ページ単位で進む |
| **Ctrl+←** / **Shift+←** | ページ単位で戻る |
| **Ctrl+.** | 次のチャプターへ |
| **Ctrl+,** | 前のチャプターへ |
| **Ctrl+A** | 現在位置にチャプターを追加 |
| **[** | クリップ開始位置を設定 |
| **]** | クリップ終了位置を設定 |

### 音量

| キー | 機能 |
|------|------|
| **↑** | 音量アップ |
| **↓** | 音量ダウン |
| **Ctrl+↑** | ミュート解除 |
| **Ctrl+↓** | ミュート |

### 編集・管理

| キー | 機能 |
|------|------|
| **Ctrl+T** | タグメニューを開く |
| **Ctrl+E** | タグ入力欄にフォーカス（再押しでプレイヤーに戻る） |
| **Ctrl+M** | メディアプレイヤーにフォーカス |
| **Ctrl+D** | 現在のファイルをDBから削除 |
| **Ctrl+S** | DBを保存 |
| **Ctrl+L** | プレイヤーのロック切り替え |
| **Ctrl+R** | リピート再生 ON/OFF |

## サポートファイル形式

再生可能な形式はシステムにインストールされているコーデックに依存します（Qt QMediaPlayer使用）。

### 動画

`.mp4` `.avi` `.mkv` `.mov` `.wmv` `.webm` `.flv` `.m4v`

### 音声

`.mp3` `.aac` `.flac` `.wav` `.ogg` `.wma` `.m4a`

### 画像

`.jpg` `.jpeg` `.png` `.gif` `.bmp` `.webp`

---

## 動作環境

| 項目 | 要件 |
|------|------|
| **OS** | Windows 10 / 11 (64-bit) |
| **メモリ** | 4GB以上推奨 |
| **ディスク** | 100MB以上の空き容量 |

### 外部ツール

| ツール | 用途 | 同梱 |
|--------|------|------|
| **yt-dlp** | YouTube等の動画ダウンロード | 同梱済み |
| **FFmpeg** | 動画変換・サムネイル生成 | 要設定（[FFmpegガイド](FFMPEG_GUIDE.md)参照） |
| **Python Embeddable** | AI類似検索 | 自動セットアップ |
